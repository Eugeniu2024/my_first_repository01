# Podskazki po komandnoi stroke

Komanda smeni directorii
```sh
cd c:\folder_name
```

Komanda otobrajenie tekusei directorii.
macos,Linux
```sh
pwd
```
Listing tekusei directorii
Windows:
```sh
dir
```
Linux,Macos:
```sh
ls
```

Udalenie faila v Windows:
```sh
del <filename>
```
V Linux,Macos:
```sh
rm <filename>
```
